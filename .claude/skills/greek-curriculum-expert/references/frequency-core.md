# High-Frequency Core

The 40% of Greek that covers 90% of casual spoken production. Everything here should be drillable before anything outside this list is considered.

### Conjugation families explained

For data modelling: Greek verbs fall into a small number of conjugation families that determine their present-tense endings. This is the primary classification axis.

- **group1** (-ω verbs): γράφω, δουλεύω, μένω. Present: -ω, -εις, -ει, -ουμε, -ετε, -ουν. Largest group.
- **group2a** (-άω/-ώ verbs): μιλάω, αγαπάω. Present: -άω/-ώ, -άς, -άει/-ά, -άμε/-ούμε, -άτε, -άνε/-ούν. Stress on ending.
- **group2b** (-ώ contracted verbs): μπορώ, οδηγώ. Present: -ώ, -είς, -εί, -ούμε, -είτε, -ούν.
- **irregular**: είμαι, έχω, λέω, τρώω. Individual patterns.
- **deponent**: έρχομαι, κοιμάμαι. Passive endings, active meaning.

The aorist stem is independent of conjugation family and must be stored per-verb.

### Production note on neuter nouns
Notice that neuter nouns have identical nominative and accusative. This is a huge win for learners: neuter nouns are "free" in accusative production. Prioritise masculine and feminine nouns for accusative drilling since those are where the form changes.

## Tier 1 Constructions (pattern slots)

These are the bridge between chunks and free production:

| Pattern | Slot types | Example |
|---------|-----------|---------|
| Θέλω + noun(acc) | any accusative noun | Θέλω έναν καφέ |
| Θέλω να + verb(subj) | any subjunctive verb | Θέλω να φάω |
| Μπορώ να + verb(subj) | any subjunctive verb | Μπορώ να πάω |
| Πρέπει να + verb(subj) | any subjunctive verb | Πρέπει να δουλέψω |
| Θα + verb(subj) | any subjunctive verb | Θα πάω αύριο |
| verb + noun(acc) | transitive verb + object | Βλέπω τον φίλο μου |
| verb + pronoun(acc) + verb | clitic pronoun | Τον βλέπω |
| Μου αρέσει + noun(nom) | any nominative noun | Μου αρέσει ο καφές |
| noun(nom) + adj(nom) | gender-matched pair | Ο καφές είναι ζεστός |

## What's NOT in the core

Conspicuously absent from the high-frequency core:

- **Passive voice**: deponent verbs (θυμάμαι, κοιμάμαι) are in the core as individual verbs, but passive as a system is not
- **Genitive case**: only for pronouns (μου/σου/του) and possession; not as a general case
- **Subjunctive as a concept**: it appears in patterns (να + verb) but isn't taught as a mood
- **Adjective declension**: adjectives appear but only in nominative agreement; case agreement deferred
- **Numbers beyond 1-10**: χίλια is not high-frequency in production
- **Formal register**: everything here is casual/neutral spoken Greek
