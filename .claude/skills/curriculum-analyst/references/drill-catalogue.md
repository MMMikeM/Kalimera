# Drill Catalogue

Valid drill types that build retrieval speed. All require production, all should be timed.

## Core Drill Types

### 1. Translation Sprint

**What:** Full sentence English → Greek translation
**Why:** Most direct simulation of speaking (thought → Greek)
**Timer:** 5-7 seconds depending on length

| Prompt | Expected Response |
|--------|-------------------|
| I see the child | Βλέπω το παιδί |
| She wants coffee | Θέλει καφέ |
| We go to the school | Πηγαίνουμε στο σχολείο |

**Progression:**
- Start with 3-4 word sentences using known vocabulary
- Increase length gradually
- Mix tenses once present is automatic

---

### 2. Conjugation Drill

**What:** Produce a specific conjugation from infinitive + subject
**Why:** Isolates verb form retrieval
**Timer:** 3-4 seconds

| Prompt | Expected Response |
|--------|-------------------|
| θέλω → they | θέλουν |
| πηγαίνω → we | πηγαίνουμε |
| βλέπω → she | βλέπει |

**Variations:**
- Present tense forms
- Aorist (past) forms
- Mixed tense (prompt includes tense marker)

---

### 3. Case Transformation

**What:** Transform nominative to required case
**Why:** Builds automatic case application
**Timer:** 3-4 seconds

| Prompt | Expected Response |
|--------|-------------------|
| ο φίλος → accusative | τον φίλο |
| η γυναίκα → genitive | της γυναίκας |
| το παιδί → accusative | το παιδί |

**Variations:**
- Article + noun together
- Just the article
- Just the noun ending

---

### 4. Article Selection

**What:** Produce correct article for gender + case combination
**Why:** Articles are high-frequency and must be instant
**Timer:** 2-3 seconds

| Prompt | Expected Response |
|--------|-------------------|
| masc + accusative | τον |
| fem + genitive | της |
| neuter + nominative | το |

**Progression:**
- Definite articles first
- Add indefinite articles
- Add plural forms

---

### 5. Pronoun Insertion

**What:** Insert correct weak object pronoun
**Why:** Pronoun placement and form must be automatic for fluency
**Timer:** 3-4 seconds

| Prompt | Expected Response |
|--------|-------------------|
| I see ___ (him) | τον |
| She gives ___ (to me) the book | μου |
| We want ___ (them, masc) | τους |

**Variations:**
- Direct object pronouns
- Indirect object pronouns
- Combined (genitive + accusative)

---

### 6. Gap Fill (No Options)

**What:** Complete sentence with missing element, no word bank
**Why:** Forces retrieval without recognition crutch
**Timer:** 4-5 seconds

| Prompt | Expected Response |
|--------|-------------------|
| Θέλω ___ καφέ (the, accusative) | τον |
| Πηγαίνουμε ___ σχολείο (to the) | στο |
| ___ βλέπω (I see him) | Τον |

**Critical:** No multiple choice. Blank input only.

---

### 7. Preposition + Case

**What:** Produce correct preposition + article/noun combination
**Why:** Preposition case requirements must be automatic
**Timer:** 3-4 seconds

| Prompt | Expected Response |
|--------|-------------------|
| to + the school (neut) | στο σχολείο |
| from + the woman | από τη γυναίκα |
| with + the friend (masc) | με τον φίλο |

---

### 8. Verb + Object Agreement

**What:** Produce verb with correctly-cased object
**Why:** Integrates verb retrieval with case application
**Timer:** 5-6 seconds

| Prompt | Expected Response |
|--------|-------------------|
| I want + the coffee | Θέλω τον καφέ |
| She sees + the child | Βλέπει το παιδί |
| We have + the book | Έχουμε το βιβλίο |

---

## Timing Guidelines

| Drill Type | Timer | Rationale |
|------------|-------|-----------|
| Single word/article | 2-3 sec | Minimal retrieval |
| Short transformation | 3-4 sec | One operation |
| Short sentence | 4-5 sec | Multiple retrievals |
| Full sentence | 5-7 sec | Integration required |

**Principle:** Timer should feel slightly rushed. If user consistently beats the timer comfortably, reduce it.

## Progression Model

### Phase 1: Isolated Forms
Drill individual components (articles, pronouns, conjugations) until fast.

### Phase 2: Two-Element Combinations
Combine two components (verb + object, preposition + noun).

### Phase 3: Full Sentences
Produce complete sentences requiring multiple automatic retrievals.

### Mastery Criteria
An item is mastered when:
- Accuracy > 90% over last 10 attempts
- Average response time < target (varies by type)
- Consistent across 3+ sessions

## What Makes a Drill Invalid

| Problem | Why It Fails |
|---------|--------------|
| Multiple choice | Recognition, not production |
| Word bank | Recognition with reduced options |
| Greek → English | Wrong direction |
| First letter hint | Reduces retrieval demand |
| Visible grammar table | Bypasses memory |
| Generous timer | Allows conscious rule application |
| No timer at all | No pressure = no proceduralization |
